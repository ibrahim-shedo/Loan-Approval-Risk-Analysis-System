## 📌 Project Overview

This project is an end-to-end **Data Science & Business Intelligence system** designed to analyze and understand **loan approval decisions in a banking environment**.

It combines:
- Data preprocessing & feature engineering  
- Machine learning modeling  
- Business intelligence dashboarding  
- Financial risk analysis  

👉 The system simulates how banks evaluate **credit risk and customer eligibility**.

---

## 🎯 Business Problem

Financial institutions must decide:

> ❓ *Which customers should be approved for a loan?*  
> ❓ *What factors influence approval or rejection?*

A poor decision can lead to:
- High default risk  
- Financial losses  
- Poor customer experience  

👉 This project builds a **data-driven decision support system**.

---

## 🧠 Solution Approach

### 1️⃣ Data Processing
- Cleaned missing values using statistical imputation  
- Encoded categorical variables  
- Removed irrelevant identifiers  

---

### 2️⃣ Feature Engineering (Key Innovation)

Created domain-specific financial features:

- **Total Income**
- **Loan-to-Income Ratio** (risk indicator)
- **EMI (Estimated Monthly Installment)**
- **EMI-to-Income Ratio (Affordability)**

👉 These simulate real-world **credit scoring logic**

---

### 3️⃣ Machine Learning Model

- Model: **Logistic Regression**
- Goal: Predict Loan Approval (Y/N)

#### 📈 Performance:
- Accuracy: **81%**
- Precision: **0.78**
- Recall: **1.00**
- F1 Score: **0.88**

👉 High recall ensures **eligible customers are not rejected**

---

### 4️⃣ Business Intelligence Dashboard

Built an interactive dashboard using Tableau to translate model insights into **business decisions**.

#### 🔍 Dashboard Features:
- KPI Cards (Applications, Approval Rate, Risk %)
- Credit History Impact Analysis
- Income Distribution (Box Plot)
- Education & Location Analysis
- Loan-to-Income Risk Scatter Plot
- Interactive Filters (Gender, Education, Area, etc.)

## 📊 Key Insights

- ✅ **Credit history is the strongest predictor of loan approval**
- ❌ Income alone does NOT guarantee approval  
- ⚠️ High Loan-to-Income ratio increases rejection probability  
- 📍 Semiurban areas show slightly higher approval rates  
- 🎓 Education has moderate influence
